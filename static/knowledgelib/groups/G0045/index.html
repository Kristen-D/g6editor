menuPass

menuPass是一个威胁组织，似乎来自中国，自2009年左右开始活跃。该组织的目标是医疗保健，国防，航空航天和政府部门，至少自2014年以来一直针对日本受害者。2016年和2017年，该组织有针对性的托管IT服务提供商，制造和采矿公司以及大学。[1] [2] [3] [4] [5]

ID：G0045
别名：menuPass，Stone Panda，APT10，Red Apollo，CVNX，HOGFISH
贡献者：Edward Millington; 迈克尔考克斯

版本：1.0

使用的技术

域ID名称使用
企业T1087帐户发现menuPass使用Microsoft管理工具csvde.exe导出Active Directory数据。[6]
企业T1059命令行界面menuPass使用命令行界面和反向shell执行命令。该小组使用了修改版的pentesting脚本wmiexec.vbs来执行命令。[4] [6] [7]
企业T1090连接代理menuPass使用全球服务提供商的IP作为来自受害者的C2流量的代理。[5]
企业T1003凭证倾销menuPass使用了修改版的测试工具wmiexec.vbs和secretsdump.py来转储凭证。[6] [7]
企业T1002数据压缩menuPass在使用TAR和RAR进行渗透之前已经压缩了文件。[4] [6]
企业T1039网络共享驱动器中的数据menuPass通过安装网络共享net use并使用Robocopy传输数据，从远程系统收集数据。[4]
企业T1074数据分阶段menuPass在多部分归档中的exfiltration之前分阶段处理数据，通常保存在回收站中。[4]
企业T1140反混淆/解码文件或信息menuPass在宏中使用certutil来解码附加到电子邮件的dropper文档中包含的base64编码内容。certutil -decode当丢弃UPPERCUT时，该组用于解码受害者机器上的文件。[8] [9]
企业T1038DLL搜索顺序劫持menuPass使用了DLL搜索顺序劫持。[4]
企业T1073DLL Side-LoadingmenuPass使用DLL侧载来启动Mimikatz和PwDump6以及UPPERCUT的版本。[6] [9]
企业T1107文件删除一个menuPass宏删除文件，它已经解码和解压后他们。[8]
企业T1046网络服务扫描menuPass使用类似于Ping的 tcping.exe 来探测感兴趣的系统上的端口状态。[6]
企业T1027混淆的文件或信息menuPass使用base64在其恶意软件中编码字符串，并使用密钥0x40进行简单的单字节XOR混淆。[8] [9]
企业T1086电源外壳menuPass使用PowerSploit将shellcode注入PowerShell。[6]
企业T1093流程空心menuPass使用iexplore.exe中的进程挖空来加载RedLeaves种植体。[8]
企业T1076远程桌面协议menuPass使用RDP连接在受害者网络上移动。[4]
企业T1105远程文件复制menuPass已在受害者身上安装了更新和新的恶意软件。[4]
企业T1021远程服务menuPass使用Putty Secure Copy Client（PSCP）传输数据。[4]
企业T1018远程系统发现menuPass uses scripts to enumerate IP ranges on the victim network. menuPass has also issued the command net view /domain to a PlugX implant to gather information about remote systems on the network.[6][5]
EnterpriseT1053Scheduled TaskmenuPass has used a script (atexec.py) to execute a command on a target machine via Task Scheduler.[6]
EnterpriseT1064ScriptingmenuPass has used malicious macros embedded inside Office documents to execute files.[8][9]
EnterpriseT1193Spearphishing AttachmentmenuPass has sent malicious Office documents via email as part of spearphishing campaigns as well as executables disguised as documents.[6][5][9]
EnterpriseT1016System Network Configuration DiscoverymenuPass has used several tools to scan for open NetBIOS nameservers and enumerate NetBIOS sessions.[6]
EnterpriseT1049System Network Connections DiscoverymenuPass has used net use to conduct connectivity checks to machines.[4]
EnterpriseT1199Trusted RelationshipmenuPass has used legitimate access granted to Managed Service Providers in order to access victims of interest.[6][5]
EnterpriseT1204User ExecutionmenuPass has attempted to get victims to open malicious files sent via email as part of spearphishing campaigns.[6][5][8]
EnterpriseT1078Valid AccountsmenuPass has used valid accounts shared between Managed Service Providers and clients to move between the two environments.[4]
EnterpriseT1047Windows Management InstrumentationmenuPass uses a modified version of pentesting script wmiexec.vbs, which logs into a remote machine using WMI.[6][7]
