APT29

APT29是一个威胁集团，归功于俄罗斯政府，至少自2008年以来一直运作。[1] [2]据报道，该集团于2015年夏季开始对民主党全国委员会做出妥协。[3]

ID：G0016
别名：APT29，The Dukes，Cozy Bear，CozyDuke
版本：1.0

使用的技术

域ID名称使用
企业T1015辅助功能APT29使用粘性密钥来获取未经身份验证的特权控制台访问。[4] [5]
企业T1088绕过用户帐户控制APT29绕过了UAC。[4]
企业T1172域前端APT29使用Tor的温顺域前端插件来隐藏C2流量的目的地。[4]
企业T1203利用客户端执行APT29已经使用多种软件漏洞利用常见的客户端软件，如Microsoft Word和Adobe Reader，以获得代码执行的一部分。[1]
企业T1070主机上的指示器移除APT29使用SDelete来移除受害者的文物。[4]
企业T1188多跳代理APT29使用的后门创建了Tor隐藏服务，以将来自Tor客户端的流量转发到本地端口3389（RDP），139（Netbios）和445（SMB），从而实现从网络外部的完全远程访问。[4]
企业T1075传递哈希APT29使用Kerberos票证攻击进行横向移动。[4]
企业T1086电源外壳APT29使用上传到CozyCar安装的编码PowerShell脚本来下载和安装SeaDuke。APT29还使用PowerShell脚本来逃避防御。[6] [4]
企业T1060注册表运行键/启动文件夹APT29添加了Registry Run键以建立持久性。[4]
企业T1053预定任务APT29使用命名和被劫持的计划任务来建立持久性。[4]
企业T1064脚本APT29 has used encoded PowerShell scripts uploaded to CozyCar installations to download and install SeaDuke, as well as to evade defenses.[6][4]
EnterpriseT1045Software PackingAPT29 used UPX to pack files.[4]
EnterpriseT1193Spearphishing AttachmentAPT29 has used spearphishing with an attachment to deliver files with exploits to initial victims.[1]
EnterpriseT1192Spearphishing LinkAPT29 has used spearphishing with a link to trick victims into clicking on a link to a zip file containing malicious files.[4]
EnterpriseT1204User ExecutionAPT29 has used various forms of spearphishing attempting to get a user to open links or attachments.[1]
EnterpriseT1047Windows Management InstrumentationAPT29 used WMI to steal credentials and execute backdoors at a future time.[4]
EnterpriseT1084Windows Management Instrumentation Event SubscriptionAPT29 has used WMI event filters to establish persistence.[4]
