更新 - 2017年4月


2017年4月更新包含几项新技术，新报告和内容更改：

扩展到133种技术，增加了：

  • Netsh Helper DLL
  • 通过模块加载执行
  • 安装根证书
  • 认证包
  • 数据编码
  • 外部远程服务

基于社区输入和新信息对若干现有技术的更新，包括：

  • 绕过用户帐户控制 - 更新了说明和UACMe信息
  • 文件系统权限弱点 - 添加了可执行安装程序变体
  • Service Registry Permissions弱点 - 更新了包含系统服务帐户的说明
  • 系统固件 - 添加了CHIPSEC参考
  • 输入捕获 - 添加了VPN凭证捕获参考
  • 渗透到其他网络介质 - 添加蓝牙/火焰示例
  • 端口监视器 - 更正了注册表项参考
  • 通过API执行 - 添加了执行API列表
  • Regsvr32 - 添加了最近报告的威胁参考

集团增加：

  • Winnti集团
  • menuPass
  • FIN7
  • Gamaredon集团
  • RTM

软件增加：

  • Winnti
  • StreamEx
  • 火焰
  • ChChes
  • 能量源
  • TextMate的
  • 无齿翼龙
  • RTM
  • MoonWind
  • 在ATT和CK中使用的软件标签中添加了更多详细信息，并添加了额外的“实用程序”类别
