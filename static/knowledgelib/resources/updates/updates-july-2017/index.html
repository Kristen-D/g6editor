更新 - 2017年7月


2017年7月的最新更新.ATT和CK现在在Windows，Mac和Linux上共有169种技术：

推出适用于企业技术的Mac和Linux ATT＆CK

  • 共有89种技术，包括MacOS / OS X的29个附加功能 - Mac技术矩阵
  • 总共80项技术，包括Linux的12项补充 - Linux Technique Matrix
  • 许多现有技术还接收内容更新以包括Mac和/或Linux的细节

适用于Windows的多达140种技术：

新技术：

  • 访问令牌操作
  • 网络共享发现
  • 创建帐号
  • Office应用程序启动
  • 应用匀场
  • 反混淆/解码文件或信息
  • 私钥
  • 隐藏文件和目录

更新技术：

  • 暴力 - 添加密码喷涂
  • Trusted Developer Utilities - 从Msbuild重命名，添加了DNX，RCSI和WinDbg / CDB的示例
  • 系统网络配置发现 - 从“本地”重命名，以包含本地和远程操作
  • 系统网络连接发现 - 从“本地”重命名以包含本地和远程操作
  • 有效帐户 - 从合法凭据重命名为帐户创建帐户中的对手创建的帐户
  • 帐户操作 - 从凭证操作重命名，以更好地适应技术的意图
  • 辅助功能 - 对技术细节进行大量更新
  • 系统固件 - 基本输入/输出系统的通用名称，用于考虑（U）EFI固件持久性
  • 传递故障单 - 更新检测和缓解信息

对组的更改包括三个新组：

  • 石油钻机
  • APT32
  • FIN10

软件更改包括新的软件配置文件：

  • POSHSPY
  • HALFBAKED
  • EvilGrab
  • 红色的叶子
  • 钴罢工
  • 挡风玻璃
  • KOMPROGO
  • SOUNDBITE
  • PHOREAL
  • SNUGRIDE
  • 的certutil
  • XAgentOSX
  • KOMPLEX
  • Janicab
