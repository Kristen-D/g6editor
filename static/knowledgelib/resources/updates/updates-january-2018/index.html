三种技术改名

许多技术已更新

更改包括添加新的技术描述信息，检测和缓解详细信息，参考和对手使用示例。其中包括主要修订版本，例如Process Injection和Access Token Manipulation，以在技术说明中添加大量新信息，以及稍微修订，例如InstallUtil，以添加一些其他详细信息。

整合平台参数 - 跟踪各个OS平台版本和版本变得非常麻烦。由于所描述的许多技术适用于大多数平台版本，因此我们决定将它们合并为一个标记。任何版本要求都将在技术的技术说明和要求部分中捕获

  • MacOS / OS X - > macOS
